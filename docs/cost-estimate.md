# 月間利用料金の試算

## 前提条件

- **営業日**: 月あたり約 21日（土日祝日を除く平日のみ）
- **1日あたり**: 10件のメッセージ
- **月間合計**: 210リクエスト/月

---

## サービス別コスト

### 1. AWS Bedrock（Claude 3.5 Sonnet v2）

最大のコスト要因。

- **モデル**: `apac.anthropic.claude-3-5-sonnet-20241022-v2:0`（APAC クロスリージョン推論）
- **リージョン**: ap-northeast-1

#### 料金単価

| 項目 | 単価 |
|------|------|
| Input | $3.00 / 1M tokens |
| Output | $15.00 / 1M tokens |

#### 1リクエストあたりのトークン見積もり

| 項目 | Input tokens | Output tokens |
|------|-------------|---------------|
| System prompt（約700字） | ~500 | - |
| ユーザーメッセージ | ~100 | - |
| Tool定義（4ツール） | ~1,500 | - |
| AI応答（ツール呼び出し） | - | ~300 |
| Tool result | ~500 | - |
| AI最終応答 | - | ~200 |
| **合計** | **~2,600** | **~500** |

※ Tool Useループが平均2回実行される想定

#### 月間計算

- Input: 2,600 × 210 = 546,000 tokens → $3.00 × 0.546 = **$1.64**
- Output: 500 × 210 = 105,000 tokens → $15.00 × 0.105 = **$1.58**
- **合計: 約 $3.22/月**

---

### 2. AWS Lambda

- **構成**: メモリ 256MB、タイムアウト 60秒
- **リクエスト数**: ~630回/月（events + interactive + 内部呼び出し）
- **実行時間**: 630 × 10秒 × 0.25GB = 1,575 GB-秒/月
- **無料枠**: 100万リクエスト/月 + 400,000 GB-秒/月
- **合計: $0/月（無料枠内）**

---

### 3. Amazon API Gateway（HTTP API）

- **料金**: $1.00 / 100万リクエスト
- **月間リクエスト**: ~630回
- **合計: $0/月（無料枠内）**

---

### 4. Amazon DynamoDB（オンデマンド）

- **テーブル**: conversations, oauth_tokens（PAY_PER_REQUEST）
- **書き込み**: 210 × 3 = 630 WRU/月 → ~$0.001
- **読み込み**: 210 × 3 = 630 RRU/月 → ~$0.0002
- **合計: 約 $0/月（ほぼ無視できる金額）**

---

### 5. AWS Secrets Manager

- **シークレット数**: 2（slack-secrets, google-secrets）
- **固定費**: $0.40 / シークレット / 月
- **API呼び出し**: ~630回/月 → ~$0.003
- **合計: 約 $0.80/月**

---

### 6. Amazon CloudWatch Logs

- **料金**: $0.76 / GB（データ取り込み）
- **ログ量**: 210リクエスト × ~2KB = ~0.42MB/月
- **保持期間**: 14日
- **合計: 約 $0/月（ほぼ無視できる金額）**

---

### 7. Google Calendar API

- 無料で利用可能
- 1日あたりの上限: 1,000,000リクエスト
- **合計: $0/月**

---

### 8. Slack API

- Slack Bolt/API の利用自体は無料
- ワークスペースのプラン費用は別途
- **合計: $0/月**

---

### 9. GCP（Google Cloud）

- OAuth用クライアント設定のみ
- **合計: $0/月**

---
